# Easy Chat - Backend

Backend API for Easy Chat application. Handles Gemini API integration and serves chat responses.

## Tech Stack
- Node.js
- Express.js
- Google Gemini API

## Related Repositories
- Frontend: https://github.com/vivek7941/EasyChat-frontend

## Setup

Install dependencies:
```bash
npm install
```

Create `.env` file:
```
GEMINI_API_KEY=your_gemini_api_key

```

Run:
```bash
npm start
```
